# CodeAlpha_IMDB_Sentiment_Analysis
# IMDB Movie Reviews Sentiment Analysis

## Project Overview

This project analyzes movie reviews from the IMDB dataset and determines sentiment patterns using Natural Language Processing techniques.

## Objectives

* Analyze review sentiments
* Explore positive and negative opinions
* Visualize sentiment distributions
* Understand customer feedback patterns

## Tools Used

* Python
* Pandas
* TextBlob
* WordCloud
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset

IMDB Movie Reviews Dataset (50,000 Reviews)

## Key Findings

* Dataset contains 25,000 positive and 25,000 negative reviews.
* No missing values were found.
* Positive reviews were slightly longer on average.
* Word clouds revealed commonly used positive and negative terms.
* Sentiment polarity scores provided insight into emotional intensity.
